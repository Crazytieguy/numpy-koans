# NumPy Koans

Learn NumPy through 132 interactive exercises in your browser. No installation needed.

**https://crazytieguy.github.io/numpy-koans/**

Fill in the blanks, starting from the basics and progressing to advanced techniques. Real Python and NumPy run in your browser via [Pyodide](https://pyodide.org).

## Topics

- Array Creation
- Array Properties
- Indexing & Slicing
- Reshaping & Manipulating
- Math Operations
- Aggregations & Statistics
- Broadcasting
- Boolean & Fancy Indexing
- Linear Algebra
- Random & Sampling
- Advanced Techniques
- Performance & Patterns

## Running locally

Open `index.html` in a browser, or serve it:

```
python -m http.server 8765
```
